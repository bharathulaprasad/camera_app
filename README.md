

Requirements:
Standard Raspberry Pi preferrably 4 or 5 with Camera bundle, or externally connected via USB
python3.13  with flask, cv2 with basic knowledge on routing URL

Features:
Video recording support
Motion detection
Image or Video Saving upon motion detection
Web interface


Future improvements:
Tunneling so that its available on web like cloudflared tunnel --url http://localhost:port
Proposed use of Gunicorn for production ready, python venv with requirements, so that evironment is isolated with external python stuff.
